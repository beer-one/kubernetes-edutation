```shell
$ brew install hyperkit

```



## minikube start

```shell
# Intel Mac OS
$ minikube start --vm-driver=hyperkit \
    --kubernetes-version=v1.25.3 \
    --cpus='2' \
    --memory='8g'

# Apple slicon
$ minikube start --vm-driver=qemu \
    --kubernetes-version=v1.25.3 \
    --cpus='2' \
    --memory='8g'
```
